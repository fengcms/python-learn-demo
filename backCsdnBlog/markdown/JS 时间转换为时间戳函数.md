在实际的项目中，一般都是数据库中存储的是时间戳，而页面上根据需要转换为时间。但是后端的同学直接写了一个时间存储了。给我的也是时间值。这我郁闷了，通过查阅资料，于是写了一个函数。


```
//时间格式 2015-12-05 00:00:00
function getTimeStamp (time){
	return new Date(time).getTime()/1000;
}
```
测试成功。
