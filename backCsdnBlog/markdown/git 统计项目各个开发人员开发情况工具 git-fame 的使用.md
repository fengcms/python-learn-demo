#git 统计项目各个开发人员开发情况工具 git-fame 的使用

作为管理人员，我们需要统计一下小组内各个开发人员的详细开发情况，使用 git 可以很方便的统计，但是怎么统计，这是个问题。

今天我找到了一个工具 `git-fame` 使用非常方便。

```#
#install
sudo gem install git-fame
```

安装完成后，在项目中使用 `git fame` 命令，就可以看到统计情况了，结果如下：

```#
$ git fame
```
